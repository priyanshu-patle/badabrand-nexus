# Demo Screenshot Guide

Use these routes to prepare marketplace screenshots.

## Public Website Screens

- Homepage: `/`
- Services: `/services`
- Marketplace: `/marketplace`
- Pricing: `/pricing`
- Portfolio: `/portfolio`
- Blog: `/blog`
- Contact: `/contact`

## Dashboard Screens

- Admin dashboard: `/admin`
- Admin content hub: `/admin/content`
- Admin system settings: `/admin/settings`
- Admin marketplace: `/admin/marketplace`
- Admin payments: `/admin/payments`
- Client dashboard: `/client`
- Client invoices: `/client/invoices`
- Client payments: `/client/payments`
- Client support: `/client/tickets`

1. Homepage hero and services

4. Admin dashboard overview
