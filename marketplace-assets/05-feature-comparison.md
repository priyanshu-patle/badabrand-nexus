# Feature Comparison

This file helps position the product in the marketplace listing.

## Included In This Package

- Public corporate website
- Customer portal
- Developer dashboard
- Admin dashboard
- Services and pricing management
- Marketplace product sales
- Invoices and manual payment approvals
- Support tickets and replies
- Project tracking
- Proposal and contract workflows
- CMS and branding controls
- Legal pages, blog, FAQ, careers, portfolio, and team management

## Strong Selling Points

- One codebase for public website plus dashboards
- Admin-manageable content instead of hardcoded sections
- Suitable for IT agencies, hosting providers, service studios, and digital product sellers
- XAMPP and shared-hosting friendly structure
- Clean dark interface with responsive layouts
