# Feature Reference

This document lists the major working modules inside the platform.

## Public Website Features

- Responsive homepage
- Services page
- Pricing page
- Marketplace page
- Contact page
- About page
- Legal pages
- Blog section structure
- Careers section structure
- FAQ section structure
- WhatsApp floating action button

## Customer Portal Features

- Secure login
- Terms acceptance on first login
- Dashboard summary
- Orders and services list
- Marketplace ordering
- Invoice list
- Printable invoice view
- Text invoice export
- Payment submission
- QR and bank transfer flow
- Support ticket creation
- Support conversation view
- Project tracker
- Proposal submission
- Proposal document view
- Contract document view
- Contract signing
- File download area

## Admin Features

- Dashboard analytics summary
- CMS and branding tools
- Services and plans manager
- Marketplace product manager
- User management
- Payment approval
- Invoice creation and editing
- Support response tools
- Marketing tools
- Project and contract management
- Proposal management
- Documentation page

## Developer Features

- Developer dashboard
- Assigned projects
- Support queue

## Billing Features

- Order creation
- Receipt number generation
- Invoice generation
- Invoice items
- GST amount handling
- Invoice status flow
- Payment proof upload
- Payment review states:
  - `pending`
  - `approved`
  - `rejected`
  - `cancelled`
  - `refund`
  - `invalid`

## Current Limitations

- Not every static content page is fully CRUD-managed yet
- PDF generation is mainly browser print/save flow
- Some advanced modules still use compact internal workflows rather than a dedicated CRUD section
