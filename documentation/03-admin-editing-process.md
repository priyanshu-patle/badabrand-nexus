# Admin Editing Process

This document explains how content and business modules are currently edited from the admin panel.

## CMS And Branding

Path: `/admin/cms`

Editable here:

- Site title
- Footer company name
- Hero title
- Hero subtitle
- Primary CTA
- Secondary CTA
- About summary
- Footer text
- Contact phone
- Contact email
- Contact address
- Logo upload
- Favicon upload

## Homepage Highlights

Also edited in `/admin/cms`

- Add highlight
- Edit highlight
- Delete highlight
- Control badge, title, subtitle, CTA label, CTA link, and sort order

## Homepage Stats

Also edited in `/admin/cms`

- Add stat
- Edit stat
- Delete stat
- Control label, value, suffix, and display order

## Testimonials

Also edited in `/admin/cms`

- Add testimonial
- Edit testimonial
- Delete testimonial
- Control name, role, quote, and sort order

## Services And Plans

Path: `/admin/services`

- Add service
- Edit service
- Delete service
- Add pricing plan
- Edit pricing plan
- Delete pricing plan

## Marketplace

Path: `/admin/marketplace`

- Add product
- Edit product
- Delete product
- Upload thumbnail
- Change price, version, status, description, features, and delivery link

## Users

Path: `/admin/users`

- Add user
- Edit user
- Approve user
- Suspend user
- Delete user

## Payments And Billing

Path: `/admin/payments`

- Search payment data
- Review screenshot proof
- Change payment status
- Add invoice
- Open invoice editor

## Projects And Contracts

Path: `/admin/projects`

- Update order status
- Update project progress
- Delete order
- Add project update
- Create proposal
- Edit proposal
- Create contract
- Edit contract

## Marketing Tools

Path: `/admin/marketing`

- Send broadcast
- Send push/email style admin alerts
- Create coupons
- Review referral balances
- Review email logs
