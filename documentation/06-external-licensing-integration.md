# Deployment Notes

Badabrand Technologies now runs as a standalone product without any license activation step.

## What to configure after install

- app URL
- database credentials
- company email and phone
- WhatsApp number
- SMTP details
- SEO defaults
- social links
- branding and CMS content

## Admin startup flow

1. Install the project.
2. Import the database.
3. Login with the seeded admin account.
4. Open CMS, Content Hub, and System Settings.
5. Update branding, menus, services, products, pricing, and contact details.
