# Admin And Customer Portal Guide

## Admin panel overview

The admin panel is the operations center for the full platform.

Main areas:

- Users
- Services
- Marketplace
- Billing
- Projects
- Support
- Marketing
- Modules
- Appearance
- Content
- Settings
- Documentation
- Package Center

## Core admin workflows

### Users

- create users
- update profile and status
- review roles
- inspect activity

### Services and pricing

- add/edit services
- manage pricing plans
- maintain service categories

### Marketplace

- manage products
- approve vendor-related product items
- review vendor applications and payouts
- inspect marketplace orders and reviews

### Billing

- generate invoices
- review payment submissions
- approve or reject payment proofs
- monitor transaction history

### Projects and delivery

- manage orders as delivery items
- update progress
- review proposals
- manage contracts

### Support

- inspect tickets
- reply to customer issues
- keep response queues healthy

## Customer portal overview

Customer users can:

- order services and marketplace items
- review invoices
- upload payment proof
- open support tickets
- track projects
- review proposals
- sign contracts
- download files
- see referral data

## Vendor portal overview

Approved vendors can:

- manage their own products
- see their orders
- review payouts
- update vendor profile and payout details

## Navigation notes

- admin uses the dashboard shell with sidebar, compact topbar, and theme support
- customer and vendor portals use the same shell system for consistency
- all counts and key widgets should use real records where available
