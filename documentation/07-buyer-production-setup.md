# Buyer Production Setup

Use this checklist for the sellable buyer package.

## Before delivery

- confirm branding, logo, and favicon defaults
- confirm menu order
- confirm admin/sidebar order
- confirm responsive layout
- confirm documentation visibility
- confirm system settings and SMTP

## Production configuration

- set `APP_URL`
- set database credentials
- import the SQL file
- configure support email, phone, and WhatsApp
- update SMTP, SEO, and social links

## Important rule

Badabrand Technologies is now a standalone commercial-ready portal.

Use the admin panel to manage:

- users
- services and pricing
- marketplace products
- payments and invoices
- support tickets
- projects, proposals, and contracts
- CMS, branding, and content
