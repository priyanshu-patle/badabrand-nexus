# Badabrand Technologies v15 Release Notes

## Release summary

Badabrand Technologies `v15` is the GitHub and delivery-ready release of the platform with a polished admin experience, production-safe installer defaults, improved documentation, and a clean buyer package structure.

## Included in this release

- modernized admin workspace UI
- simplified primary navigation
- compact search-first topbar
- real-data business dashboard widgets
- module manager improvements
- premium multi-step installer
- vendor and referral system support
- public documentation and installation guides
- GitHub-ready README and listing content
